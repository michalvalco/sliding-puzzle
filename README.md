# Sliding Puzzle

A calm, cozy sliding puzzle game — beautiful photo puzzles where **every board is guaranteed solvable**. A fresh Daily Challenge each day, your own photos as puzzles, three difficulties, achievements, themes, and a relaxing soundtrack.

**▶ [Play info & screenshots](https://michalvalco.github.io/sliding-puzzle/)**  ·  **⬇ [Download for Windows](https://github.com/michalvalco/sliding-puzzle/releases/latest)**

This repository hosts the **landing page** and **public downloads** for the game.

## Download & run (Windows 10/11, 64-bit)

1. Download `SlidingPuzzle-Windows.zip` from the [latest release](https://github.com/michalvalco/sliding-puzzle/releases/latest).
2. Unzip it anywhere.
3. Run `SlidingPuzzle.exe`. First launch, Windows SmartScreen may warn about an unsigned app — choose *More info → Run anyway*.

Your progress saves next to the game. No installer, no account, nothing uploaded.

## Try it in your browser

The [landing page](https://michalvalco.github.io/sliding-puzzle/) has a live, playable mini-puzzle plus real screenshots. `sliding-puzzle.html` in this repo is a single, self-contained copy you can open or share offline.
